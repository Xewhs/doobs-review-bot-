# Doobs Staff Bot

Custom Discord bot for Doobs Uploading Services.

## Commands
- /setup-reviews
- /request-review
- /timeout
- /untimeout
- /kick
- /ban
- /unban
- /purge
- /warn
- /userinfo

STAFF_ROLE_ID
